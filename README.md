# Budgeting AI Assistant

An AI-powered personal finance chatbot built for college students.

## About
This chatbot helps college students manage their money by providing 
personalized budgeting advice, tips on saving, and guidance on 
managing student loans and credit cards.

## Features
- Personalized budget breakdowns
- 50/30/20 rule guidance
- Student loan and credit card advice
- Spending and saving tips
- Judgment-free financial coaching
